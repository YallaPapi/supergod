# Daily Authority Content Plan (i2v + Instagram + Reddit)

## Objective
Generate inbound leads for a **content creation + posting service** and close the first 10 paying clients as fast as possible.

## Core Offer (keep this fixed for 30 days)
- **Offer name:** Content Engine Sprint (30 days)
- **Who it is for:** founders/operators who need consistent short-form content but do not have a production system.
- **What they get:**
  - 20 short-form videos (reels/shorts format)
  - captions + hooks + CTAs
  - posting + scheduling support
  - weekly performance review
- **Risk reversal:** 14-day checkpoint: if no qualified inbound conversations, extend service 2 weeks at no charge.

## Lead Magnets Mapped to Offer
1. **Lead Magnet A (Top of funnel):** 30-Day Content Calendar for [niche]
- Format: Notion/Google Sheet template
- CTA: "Comment `CALENDAR` and I’ll DM it"
- Bridge to offer: shows planning gap your service fills.

2. **Lead Magnet B (Mid funnel):** 50 High-Converting Hooks Swipe File
- Format: PDF
- CTA: "DM `HOOKS`"
- Bridge to offer: demonstrates messaging quality + positions done-for-you scripting.

3. **Lead Magnet C (Bottom funnel):** Content Engine Audit (Loom, 10 minutes)
- Format: personalized teardown
- CTA: "Reply `AUDIT` for a free teardown"
- Bridge to offer: direct pre-sales diagnostic.

## Channel Strategy
- **Instagram:** primary authority + trust surface (daily reels + stories).
- **Reddit:** demand capture + credibility via helpful comments and non-spam education.
- **Rule:** one idea/day, platform-native execution on both channels.

## Evidence-Based Guardrails
- Reddit’s own Reddiquette uses a 9:1 rule of thumb for self-promo (roughly 10% self-promotional content); stay contribution-first.
- Reddit spam policy explicitly flags repeated unsolicited mass posting and repetitive financial-gain posting; rotate content and prioritize genuine participation.
- Instagram has publicly emphasized creator best-practice guidance around recommendation-friendly content and engagement signals; use short, high-retention reels and clear CTAs.

## Daily Execution System (Mon-Sat)
1. **Research (20 min):**
- Pull 10 pain points from Reddit threads in target niches.
- Pick 1 pain point with clear buying intent.

2. **Production with i2v (60-90 min batch):**
- Script 3 reels/day from one core idea:
  - Reel 1: problem agitation
  - Reel 2: tactical framework
  - Reel 3: case/example
- Render using i2v templates (face/caption-first style).
- Export platform-safe cuts (9:16, caption burned).

3. **Posting:**
- Instagram:
  - 1 reel/day
  - 3-5 story frames/day (poll + objection + CTA)
- Reddit:
  - 1 value post/day in relevant subreddit (if allowed)
  - 10-20 high-quality comments/day across threads

4. **Lead Capture + Follow-up (30-45 min):**
- DM everyone who comments keyword CTA within 2 hours.
- Send magnet first, ask 1 qualifying question, then offer audit call.

## Weekly Content Pillars (repeat)
- **Pillar 1: Mistakes** (what founders do wrong)
- **Pillar 2: Frameworks** (simple repeatable system)
- **Pillar 3: Proof** (before/after, teardown, wins)
- **Pillar 4: Objections** (time, cost, fear of cringe)
- **Pillar 5: Behind-the-scenes** (your process + automation)

## Hook Library (Instagram + Reddit Adaptations)
1. "Your content isn’t failing because of quality. It’s failing because of this distribution mistake."
2. "If I had to get clients from zero in 14 days, I’d do this exact posting stack."
3. "Most agencies sell volume. Clients actually pay for this one outcome."
4. "I reviewed 50 founder accounts. Here’s the pattern killing inbound leads."
5. "Steal this 3-part reel format if you sell a service."
6. "This took me 22 minutes to produce and generated X qualified replies."
7. "Before you post again, fix this CTA leak."
8. "No audience? Good. Use this authority loop instead."

## CTA Framework by Funnel Stage
- **Top funnel CTA:** save/share/comment keyword (low friction)
  - Example: "Comment `CALENDAR` and I’ll send the template."
- **Mid funnel CTA:** DM keyword + micro-qualification
  - Example: "DM `HOOKS` and tell me your niche; I’ll send the closest version."
- **Bottom funnel CTA:** audit/call booking
  - Example: "If you want me to map your next 14 posts, reply `AUDIT`."

## Reddit Playbook (Inbound Without Getting Flagged)
- Subreddit selection criteria:
  - active in last 24h
  - clear business/content questions
  - rules permit link/resource sharing
- Comment structure:
  - line 1: direct answer
  - line 2-4: tactical steps
  - line 5: optional soft CTA ("I can share template if useful")
- Posting ratio target:
  - at least 9 value interactions per 1 promotional mention
- Never mass-crosspost the same copy.

## 14-Day Sprint Calendar (Fastest Path to First Clients)
- **Days 1-3:** publish mistake/framework content + magnet CTA; build response velocity.
- **Days 4-7:** publish proof/teardown content; start free audits for warm leads.
- **Days 8-10:** publish objection-handling content; invite leads to short calls.
- **Days 11-14:** publish offer breakdown + urgency (limited client slots this month).

## Inbound SOP (DM to Call)
1. Trigger: keyword comment/DM.
2. Reply within 2 hours: deliver magnet.
3. Qualify with 3 questions:
- niche?
- current posting frequency?
- monthly revenue band?
4. If fit: send audit invite + booking link.
5. On call: diagnose current content system, prescribe 30-day sprint, close.

## KPI Targets (First 30 Days)
- 30 Instagram reels posted
- 300+ Reddit value comments
- 90+ keyword CTA responses
- 30+ qualified DM conversations
- 15+ audit calls booked
- 10 paying clients closed

## Optimization Rules (Do weekly)
- Keep top 20% hooks; discard bottom 50% by retention/comments.
- Double down on lead magnets with highest DM-to-call conversion.
- If a subreddit underperforms for 7 days, replace it.
- If DM response time >2h, reduce posting volume and prioritize conversion ops.

## Implementation Notes for Your Stack
- Use i2v to batch-produce 3-5 short videos per content idea and maintain daily consistency.
- Use Geelark/Instagram infra to schedule at consistent windows and avoid missed days.
- Use Reddit warmup/poster tools for account health, but keep behavior human-paced and value-first to avoid spam signals.
