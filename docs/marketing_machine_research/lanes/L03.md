# L03 — Content Engine (Hooks, Scripts, Repurposing Loops)

## 1) Three in-lane options

### Option A: Signal-First Short-Form Engine (Hook -> Script -> CTA)
Design every asset around platform recommendation signals (watch time, shares/sends, likes), then produce native short-form variants per channel.

Why this can win first 10 clients fast:
- Small businesses already perceive short-form as highest-ROI format, so sell a clear "weekly short-form growth system" offer.
- Operationally simple: one hook bank + one script bank + per-platform rendering rules.

Evidence (with date):
- TikTok says recommendations are based on user interactions (watch/like/share/search), accounts followed, and regional signals. Evidence date: December 20, 2022. https://newsroom.tiktok.com/en-us/learn-why-a-video-is-recommended-for-you
- YouTube says recommendation satisfaction signals include clicks, watch time, shares, likes/dislikes. Evidence date: September 15, 2021. https://blog.youtube/inside-youtube/on-youtubes-recommendation-system/
- Instagram launched Best Practices in Professional Dashboard including guidance on capturing attention, reels length, and algorithm/reach learning. Evidence date: October 1, 2024. https://about.fb.com/news/2024/10/best-practices-education-hub-creators-instagram/

### Option B: Response-Led Engine (Comments/Questions -> New Shorts)
Build content directly from audience comments, FAQs, and objections. Each post seeds the next post, creating compounding script inputs.

Why this can win first 10 clients fast:
- Faster ideation than “blank page” scripting.
- Converts better for services because answers naturally surface buyer objections.

Evidence (with date):
- YouTube Shorts supports replying to comments with a Short, enabling comment-driven content loops. Evidence date: August 1, 2023. https://blog.youtube/news-and-events/6-new-youtube-shorts-creation-tools/
- TikTok recommendation inputs explicitly include comments and searches, supporting response-led topic iteration. Evidence date: December 20, 2022. https://newsroom.tiktok.com/en-us/learn-why-a-video-is-recommended-for-you
- YouTube recommendation system uses shares/likes/dislikes as satisfaction signals, reinforcing response content that earns interaction. Evidence date: September 15, 2021. https://blog.youtube/inside-youtube/on-youtubes-recommendation-system/

### Option C: Pillar-to-Micro Repurposing Factory (1 long source -> multi-platform shorts)
Capture one weekly pillar source (client interview, founder rant, case breakdown), then atomize into hooks + scripts + short variants for IG/TikTok/YouTube.

Why this can win first 10 clients fast:
- Highest throughput per production hour.
- Easiest to package/pricing-anchor for service clients ("X source assets -> Y posts/week").

Evidence (with date):
- YouTube added tools to transform horizontal videos into Shorts (recomposition, layout/zoom/crop), supporting direct repurposing workflows. Evidence date: August 1, 2023. https://blog.youtube/news-and-events/6-new-youtube-shorts-creation-tools/
- Instagram Trial Reels data: after trying trial reels, 40% posted reels more often; among those, 80% saw increased non-follower reach (internal analysis note provided by Meta). Evidence date: June 12, 2025. https://about.fb.com/news/2025/06/inspiring-creativity-that-brings-people-together/
- HubSpot 2026 State of Marketing (updated Nov 18, 2025): short-form video ranked highest ROI format (48.6%). Evidence date: November 18, 2025. https://blog.hubspot.com/marketing/content-trends-global-preferences

## 3) Key failure modes + mitigations

1. Failure mode: Output feels generic/reposted; recommendations suppressed.
- Mitigation: Require original POV layer in every script (operator take, client story, example). Enforce "no raw repost" QA gate.

2. Failure mode: High volume, weak hooks -> low watch time.
- Mitigation: Hook QA rule: first 2 seconds must include pain, promise, or contrarian claim. Kill scripts that fail hook table-read.

3. Failure mode: Repurposed assets ignore platform context.
- Mitigation: Final packaging step per platform (caption style, CTA style, pacing edits). Keep master script constant, surface framing variable.

4. Failure mode: No conversion path from views to calls.
- Mitigation: Every asset mapped to one CTA objective (DM keyword, booking link, lead magnet).

5. Failure mode: Claimed performance uplifts not reproducible across niches.
- Mitigation: Mark imported benchmark claims as directional; run 2-week baseline test before pricing guarantees.
- UNPROVEN: "A fixed hook formula will outperform niche-specific creative in all verticals."
- UNPROVEN: "Daily posting is required for service-client acquisition in every market."

## 4) Concrete 14-day execution outline

Day 1:
- Define ICP and offer promise for first acquisition sprint (one niche only).
- Build 3 CTA endpoints (DM keyword, calendar link, audit form).

Day 2:
- Create hook bank: 40 hooks split into pain/proof/contrarian/story buckets.
- Draft 10 script skeletons (30-45s primary).

Day 3:
- Record 1 pillar source (20-40 min interview/rant/case breakdown).
- Extract 25 clip candidates with timestamp notes.

Day 4:
- Convert top 12 clip candidates into short scripts (platform-native openings).
- Add explicit CTA line per script.

Day 5:
- Produce/edit first 9 shorts (3 IG, 3 TikTok, 3 YouTube Shorts).
- Prepare caption variants (discovery vs conversion).

Day 6:
- Publish batch #1; activate comments/DM triage SOP.
- Start response queue (all objections/questions tagged for next scripts).

Day 7:
- Analyze first 24h/48h metrics: hold rate, completion, shares/sends, profile actions.
- Kill bottom 30%; rewrite hooks only for survivors.

Day 8:
- Produce 6 response-led shorts from comments + DMs.
- Launch batch #2.

Day 9:
- Refresh hook bank with winning patterns from days 6-8.
- Build 5 “offer proof” assets (before/after, process teardown, mini-case).

Day 10:
- Publish proof-focused batch; add stronger CTA ("DM 'PLAN' for 14-day content map").
- Start outbound reuse: send best-performing proof clips in prospect DMs/emails.

Day 11:
- Record pillar source #2 using validated winning hooks.
- Atomize into next 10-12 shorts.

Day 12:
- Publish batch #3; run A/B on first-line hook and CTA wording.
- Update script template with observed winners.

Day 13:
- Compile lane dashboard: post count, qualified DMs, booked calls, close rate.
- Turn best 3 assets into permanent onboarding examples for sales.

Day 14:
- Retrospective + scale decision:
  - Keep 2 winning content pillars.
  - Drop losing pillar.
  - Lock next 14-day production calendar.
