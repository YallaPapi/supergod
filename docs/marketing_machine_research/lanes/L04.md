# L04 Outbound Routing

## 1) Three options in-lane

### Option A: Email-first outbound, trigger to call/DM only after positive signal

**Channel design**
- Primary: cold email from warmed domain(s).
- Secondary: LinkedIn/Instagram DM only after email open + click/reply signal.

**Trigger-based routing rules**
- If prospect matches ICP and has verified work email -> enter 4-step email sequence (D1, D3, D6, D10).
- If reply contains pain/timeline/budget -> route to same-day call booking.
- If no reply after step 3 but at least 2 opens -> route to one manual LinkedIn follow-up.
- If bounce/complaint/unsub -> suppress across all channels for 90 days.

**Evidence (with date)**
- Hunter, *The State of Cold Email 2025* (published 2025): decision-makers prefer cold email as outreach channel; average campaign reply rate reported as 4.1%. https://hunter.io/the-state-of-cold-email
- Google, *New Gmail protections for a safer, less spammy inbox* (Oct 3, 2023): 2024 bulk-sender requirements (authentication, easy unsubscribe, spam threshold). https://blog.google/products-and-platforms/products/gmail/gmail-security-authentication-spam-protection/
- Yahoo Postmaster, *An Update on Enforcing Email Standards* (Dec 20, 2023): enforcement of authentication/complaint standards and one-click unsubscribe for bulk senders. https://blog.postmaster.yahooinc.com/post/737268108173230080/an-update-on-enforcing-email-standards
- FTC, *CAN-SPAM Act: A Compliance Guide for Business* (Aug 2023; edited Jan 2024): legal requirements and penalties for commercial email in the U.S. https://www.ftc.gov/business-guidance/resources/can-spam-act-compliance-guide-business

### Option B: Social DM-first (LinkedIn + Instagram), then route to call/email

**Channel design**
- Primary: manual social prospecting + personalized DMs.
- Secondary: email used only when prospect explicitly requests details/proposal.

**Trigger-based routing rules**
- If prospect engaged with your post/comment in last 14 days -> send short DM with one explicit CTA.
- If DM positive response -> route to call booking within 24h.
- If no response after 2 touches -> stop (avoid spam pattern) and move to passive nurture list.
- If prospect asks for case studies/pricing -> route to email proposal thread.

**Evidence (with date)**
- LinkedIn Help, *Prohibited software and extensions* (last updated ~2025): disallows bots/extensions that automate messaging and can restrict accounts. https://www.linkedin.com/help/linkedin/answer/a1341387
- LinkedIn Help, *Automated activity on LinkedIn* (last updated ~2025): account restrictions tied to automated activity. https://www.linkedin.com/help/linkedin/answer/a1341543
- Instagram Terms of Use page (Help Center mirror; crawled 2025): prohibits creating/accessing/collecting info in automated ways without permission. https://www.facebook.com/help/581066165581870
- Gartner press release (Jun 25, 2025): 73% of B2B buyers avoid suppliers with irrelevant outreach; relevance is critical for message routing quality. https://www.gartner.com/en/newsroom/press-releases/2025-06-25-gartner-sales-survey-finds-61-percent-of-b2b-buyers-prefer-a-rep-free-buying-experience

### Option C: Intent-first Reddit routing, then handoff to email/call

**Channel design**
- Primary: detect high-intent Reddit threads (hiring help, growth pain, content bottlenecks), respond in-thread with value.
- Secondary: move qualified prospects to DM/email only after explicit interest.

**Trigger-based routing rules**
- If post includes active pain + budget proxy (team size, launch date, workload) -> comment with actionable mini-audit; CTA = "want a 15-min teardown?"
- If commenter replies positively -> route to DM with scheduling link.
- If DM accepts call -> route to same-day qualification call.
- If no reply after 1 comment + 1 follow-up comment -> stop outreach.
- If subreddit rules block promo -> route to value-only replies and no CTA.

**Evidence (with date)**
- Reddit Help, *Spam* (updated Oct 9, 2025): bans repeated/unsolicited mass engagement (manual or automated), including unsolicited DMs. https://support.reddithelp.com/hc/en-us/articles/360043504051-Spam
- Reddit Help, *How do I keep spam out of my community?* (updated Aug 15, 2025): many communities use strict anti-promo norms (including 10% self-promo guidance in some communities). https://support.reddithelp.com/hc/en-us/articles/28012014962580-How-do-I-keep-spam-out-of-my-community
- Reddit Investor Relations, Q4/FY 2024 results (Feb 12, 2025): 101.7M DAUq and continued growth signal broad reach for niche intent capture. https://investor.redditinc.com/news-events/news-releases/news-details/2025/Reddit-Announces-Fourth-Quarter-and-Full-Year-2024-Results/default.aspx

## 2) Lane winner with explicit score math (0-100)

**Scoring framework (weighted)**
- Speed to first sales conversation: 35
- Intent quality (likelihood of real buying need): 25
- Compliance/platform risk (higher = safer): 20
- Fit with current assets (Reddit + Instagram infra + automation stack): 20

**Option scores (0-10 per criterion)**
- Option A (Email-first): speed 7, intent 6, compliance 8, asset fit 7
  - Score = `(7*35 + 6*25 + 8*20 + 7*20) / 10`
  - Score = `(245 + 150 + 160 + 140) / 10 = 695/10 = 69.5`
- Option B (Social DM-first): speed 8, intent 5, compliance 4, asset fit 8
  - Score = `(8*35 + 5*25 + 4*20 + 8*20) / 10`
  - Score = `(280 + 125 + 80 + 160) / 10 = 645/10 = 64.5`
- Option C (Reddit intent-first): speed 9, intent 8, compliance 6, asset fit 9
  - Score = `(9*35 + 8*25 + 6*20 + 9*20) / 10`
  - Score = `(315 + 200 + 120 + 180) / 10 = 815/10 = 81.5`

**Winner: Option C (81.5/100)**

Why: fastest path to first 10 paying clients is high-intent problem interception, then permissioned handoff to DM/email/call. This best matches your Reddit tooling and avoids pure cold-start inboxing.

## 3) Key failure modes + mitigations

- Failure mode: account/subreddit restrictions from spam-like behavior.
  - Mitigation: enforce hard caps (max 1 initial comment + 1 follow-up per prospect), no unsolicited mass DMs, and per-subreddit rule checks before posting.
- Failure mode: low conversion from public comment to private conversation.
  - Mitigation: use mini-audits in first reply (specific, useful, short), then single CTA to 15-minute teardown.
- Failure mode: pipeline volume volatility by subreddit traffic.
  - Mitigation: route across 8-12 subreddits by ICP; rebalance weekly by response rate.
- Failure mode: weak qualification causing time drain.
  - Mitigation: call gate requires at least 2 qualification signals (active need + timeline/budget proxy) before booking.
- Failure mode: over-automation footprint on social platforms.
  - Mitigation: keep outreach actions human-in-the-loop on LinkedIn/Instagram; use automation only for queueing, reminders, and logging.
- Failure mode: claim that Reddit intent capture consistently beats email for service sales.
  - Mitigation: mark as `UNPROVEN`; run 14-day A/B volume split (60% Reddit intent-first, 40% email-first) and compare booked-call rate.

## 4) Concrete 14-day execution outline

**Day 1-2: Setup and routing map**
- Define ICP slices (e.g., creators/coaches/ecom/local services).
- Build trigger taxonomy: pain keywords, urgency keywords, budget proxies.
- Create 3 reply templates per ICP: diagnostic, credibility, CTA.

**Day 3-4: Prospect intake + compliance guardrails**
- Build daily queue of 40 Reddit opportunities (manual + tool-assisted discovery).
- Add rule check column per subreddit (promo allowed Y/N, link rules, frequency rules).
- Define suppression list logic (no second touch after no response).

**Day 5-7: Launch wave 1**
- Execute 25-35 high-quality comments/day.
- Route positive responders to DM + 15-minute booking link.
- Send follow-up only to threads with explicit engagement.
- Target KPI by day 7: 8-12 qualified conversations started.

**Day 8-10: Add email fallback lane**
- For engaged-but-not-booked leads, send 2-step email follow-up with short proof assets.
- Keep daily Reddit comment volume steady; prioritize subreddits with best reply rate.
- Target KPI by day 10: 6-10 booked calls cumulative.

**Day 11-12: Tighten qualification + offer framing**
- Disqualify low-intent leads faster; enforce 2-signal minimum.
- Use one offer only: "content creation + posting ops sprint" with clear deliverables and start date.
- `UNPROVEN`: exact closing rate assumptions per vertical; track by segment instead of assuming.

**Day 13-14: Close sprint and route optimization**
- Run close-focused follow-ups on all qualified leads.
- Re-score routing rules from live data: comment->DM rate, DM->call rate, call->close rate.
- Keep top 3 subreddits, pause bottom 50% performers.
- End-of-day-14 target: first 3-5 paid clients closed, with sufficient pipeline to reach 10 in next cycle.
