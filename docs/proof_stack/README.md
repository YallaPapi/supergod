# Fast Proof Stack (First 10 Clients)

This proof stack is built to help you close the first 10 paying clients for content creation + posting services using:
- i2v content production
- Geelark posting automation
- Reddit warmup/posting tools
- Instagram posting infra

## What is included

1. `sample_content_packs.md`
   - Three ready-to-sell sample packs:
   - Local service lead-gen
   - Coach/consultant authority
   - Ecommerce UGC-style conversion

2. `before_after_examples.md`
   - Concrete "before vs after" transformations you can show on sales calls and in DMs.

3. `one_page_case_narrative.md`
   - One-page case-style results narrative template with measurable KPIs and automation workflow proof points.

## How to use this in 72 hours

1. Pick one niche and one sample pack.
2. Generate a 7-day mini-batch in i2v (9 short videos + 9 captions).
3. Schedule to Instagram (and Reddit where relevant) with your posting automation.
4. Export proof artifacts:
   - scheduling screenshots
   - publish log timestamps
   - post URLs
   - engagement screenshot after 48-72 hours
5. Fill the one-page case narrative with real numbers and send in outbound.
